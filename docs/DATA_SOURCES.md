## Data Accuracy Statement

All measurements and physical data in Powers Explorer are sourced from authoritative scientific databases, peer-reviewed literature, and internationally recognized standards organizations. We prioritize:

1. **Primary Sources**: Official measurements from space agencies, research institutions
2. **Current Data**: Most recent accepted values as of document date
3. **Precision Appropriate to Context**: Educational accuracy without false precision
4. **Multiple Source Verification**: Cross-reference critical values across sources
5. **Transparent Uncertainty**: Document when values are estimates or have significant uncertainty

### Precision Policy

For educational purposes, we generally use:
- **3-4 significant figures** for most measurements
- **Scientific notation** for very large or small values
- **Explicit uncertainty** for contested or estimated values
- **Rounded values** when full precision would be misleading

## Cosmic Objects Data

### Organizational System

Objects are organized by **scale level** corresponding to their approximate order of magnitude:

- **Level -35 to -10**: Subatomic and atomic scales
- **Level -9 to -1**: Microscopic scales
- **Level 0 to 3**: Human and terrestrial scales
- **Level 4 to 7**: Geographic and atmospheric scales
- **Level 8 to 12**: Solar system scales
- **Level 13 to 26**: Stellar, galactic, and cosmic scales

## Scale Levels

Scale levels are defined logarithmically based on powers of 10.

### Level Definition Methodology

Each scale level corresponds to an **exponent** in the form 10^n meters:

- **Level 0**: 10^0 m = 1 meter (human scale)
- **Level -10**: 10^−10 m = 1 Ångström (atomic scale)
- **Level 10**: 10^10 m = ~Earth's diameter scale

### Level Assignment for Objects

Objects are assigned to the level corresponding to their characteristic size:

**Example**: A hydrogen atom with radius ~5 × 10^−11 m is assigned to Level -10 (closest power of 10).

### Background Images and Themes

Each scale level has an associated visual theme:

- **Quantum scales** (-35 to -10): Abstract, wave-like patterns
- **Atomic/molecular** (-9 to -7): Atomic orbital visualizations
- **Cellular** (-6 to -4): Biological textures
- **Human scale** (0 to 3): Terrestrial landscapes
- **Planetary** (7 to 9): Space imagery
- **Stellar** (10 to 16): Nebulae, star fields
- **Galactic** (17 to 21): Galaxy imagery
- **Cosmic** (22 to 26): Deep space, cosmic web

**Image Sources**:
- NASA Image Gallery: https://images.nasa.gov/
- ESA Multimedia Gallery
- Hubble Space Telescope Archive

## Validation Methodology

### Multi-Source Verification

For critical measurements, we verify values across at least two independent authoritative sources.

**Example**: Sun's diameter
1. **NASA Sun Fact Sheet**: 1,392,700 km
2. **IAU Resolution B3 (2015)**: 1,392,684 km (nominal solar radius × 2)
3. **Used Value**: 1,392,700 km (NASA value, consistent with educational precision)

### Consistency Checks

**Dimensional Analysis**: Ensure units are consistent and calculations are correct.

**Example**: Light travel time from Earth to Moon
- Distance: 384,400,000 m
- Speed of light: 299,792,458 m/s
- Calculated time: 384,400,000 ÷ 299,792,458 = 1.282 seconds
- **Verification**: Matches published values

### Educational Rounding

For educational clarity, we may round values appropriately:

- **Full precision preserved** in data files
- **Displayed precision** appropriate to context (typically 3-4 significant figures)
- **No false precision**: Don't imply accuracy beyond source data

## Contributing Data

### Guidelines for Contributors

If you wish to add new objects or update existing data:

#### Required Information

1. **Object Name**: Official scientific name
2. **Characteristic Size**: With appropriate unit
3. **Category**: (e.g., planet, star, virus, molecule)
4. **Primary Source**: Citation to authoritative reference
5. **Secondary Source**: At least one corroborating source
6. **Uncertainty**: If applicable and known
7. **Additional Facts**: (optional) Educational facts with sources

#### Data Quality Standards

- **Authoritative Sources Only**: NASA, ESA, NIST, peer-reviewed journals, recognized textbooks
- **Current Data**: Prefer most recent measurements
- **Avoid**: Wikipedia, popular science articles, blogs (unless citing original source)
- **Cite Original Research**: If using secondary source, trace to original measurement

#### Submission Process

1. Fork the repository
2. Add/update entry in `public/assets/data/cosmic-objects.json`
3. Document sources in `docs/DATA_SOURCES.md` (this file)
4. Submit pull request with clear description
5. Maintainers will review sources and accuracy

## Known Limitations

### Uncertainty and Estimates

Some values have significant uncertainty or are estimates:

1. **Quark Size**: Theoretical point particles; any size represents interaction scale, not physical extent
2. **Galaxy Sizes**: Galaxies have diffuse boundaries; sizes are approximate
3. **Observable Universe**: Depends on cosmological model; value represents current best estimate
4. **Virus Sizes**: Vary significantly; we provide typical ranges or specific examples

### Idealized Models

Some objects are idealized for educational purposes:

- **Atoms**: Presented with Bohr radius, though quantum mechanics shows diffuse probability distributions
- **Planetary Diameters**: Given as single values though planets are not perfect spheres

### Precision vs. Accuracy Trade-off

For educational use, we balance:
- **Scientific accuracy**: Using best available data
- **Educational clarity**: Not overwhelming students with excessive precision
- **Honest representation**: Acknowledging when values are estimates

**Example**: Speed of light is exact by definition (299,792,458 m/s), but for some calculations we may display "3.0 × 10^8 m/s" for simplicity, noting it's an approximation.
